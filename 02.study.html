Python 實作

01. 從零開始：以 Python 實作經典 Hangman 遊戲之邏輯架構

學習動機與目標

受從事資訊產業的父親啟發，我自國中起便開始接觸 Python 程式設計。我非常享受程式中嚴謹的邏輯與推演過程，因此決定將其定為高中自主學習計畫的主題，利用校內外時間深入鑽研。最終，我將所學的語法實作出童年時常遊玩的吊死鬼 (Hangman) 遊戲，成功將理論轉化為具體成果。

學習動機與目標

試算表分析

03. 數據力實戰：以 Google 試算表分析「沉沒成本與稟賦效應」

學習動機與目標

在生活、課堂或社團活動中，常有製作與統整資料的需求，讓我意識到分析數據能力的重要性。因此，我規劃了一個五週自主學習計畫，學會運用Google 試算表的功能去分析表單回覆者的回覆內容。 我選擇將「沉沒成本與稟賦效應」作為我表單數據分析的主題，旨在探討影響消費者生活決策的關鍵條件與因素，將性別、年齡與各種情境設定作為分析要素。 透過這項計畫，我將同時達成兩大目標：一、學會運用 Google 試算表進行數據分析；二、培養設計有效表單內容與研究問題的能力。
